### Navigator

```fsharp
type Navigator = interface
```

```fsharp
member Invoke: (``to``: string) * (?options: NavigateOptions) -> unit
```

See the `InvokeOptions` signature below to see the construction parameters for the
`NavigateOptions` Pojo.

```fsharp
member InvokeOptions: 
    ``to``: string *
    ?resolve: bool *
    ?replace: bool *
    ?scroll: bool *
    ?state: obj
    -> unit
```

:::warning
This will replace `Invoke` in an upcoming update, as it is a superior user experience
to consume.
:::

```fsharp
member Invoke: delta: float -> unit
```
