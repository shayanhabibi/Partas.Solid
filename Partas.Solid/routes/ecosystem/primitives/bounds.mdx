# Bounds

## `ElementBounds`

Type: `interface`

- width: `int`
- height: `int`
- top: `int`
- left: `int`
- right: `int`
- bottom: `int`

## Bindings

It is suggested to set the tracking parameters to false, and only use the ones you
need. 

Depending on the circumstances, having all 3 activated can be excessive.

:::details
